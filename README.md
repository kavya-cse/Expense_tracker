# Smart Expense Tracker (CLI)

A Python-based command-line Expense Tracker that helps users manage and analyze their daily expenses. This project focuses on data analysis and user insights rather than just data storage.

## Features
- Add expense (amount, category, date)
- View all expenses
- Category-wise spending analysis
- Monthly summary of expenses
- Budget tracking with warning system
- Highest spending category detection
- Sort expenses (highest to lowest)
- Clear all data option
- Export expense report to a text file
- Input validation for accuracy

## Technologies Used
- Python
- JSON (for persistent storage)

## How It Works
- Expenses are stored in a JSON file
- Data is processed to generate insights
- Users can analyze spending patterns easily

## Purpose
This project was developed to practice data handling, analysis, and real-world problem solving using Python.
